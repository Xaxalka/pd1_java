# PD1_Java – Система тестирования

**PD1_Java** — учебный проект на Java, реализующий простую систему тестирования с графическим интерфейсом.

## Описание

Проект позволяет создавать вопросы, хранить варианты ответов и проверять ответы пользователей через GUI. Поддерживаются различные типы вопросов: одиночный выбор и множественный выбор. Система хранит пользователей и их результаты тестирования.

## Основные функции

- Создание и хранение тестовых вопросов (`TestQuestion`, `QuestionD`)
- Работа с пользователями (`User`)
- Графический интерфейс на Java Swing для взаимодействия с системой
- Поддержка различных типов вопросов
- Просмотр результатов тестирования

## Запуск проекта

1. Распаковать проект и убедиться, что используется JDK 17.
2. Открыть проект в любой IDE для Java (NeatBeans 17 IDE*,IntelliJ IDEA, Eclipse и т.д.).
3. Запустить главный класс проекта (`Main.java`) для запуска GUI.
